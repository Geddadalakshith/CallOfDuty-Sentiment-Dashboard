Call of Duty Sentiment Analysis Dashboard
Overview
This Power BI dashboard analyzes Twitter sentiment for *Call of Duty*, providing KPIs and insights into audience opinions.

Features
- Sentiment distribution (Positive, Negative, Neutral)
- Average engagement (Likes, Retweets)
- Interactive filters and visual trends

 Tools Used
- Power BI
- Twitter Sentiment Dataset (`twitter_training.csv`)
- Data Cleaning (Python)

 Files in this Repository
- `CallOfDuty_Sentiment_Dashboard.pbix` → Power BI dashboard file  
- `images/` → Screenshots for preview  
- `README.md` → This documentation file
