Call of Duty Twitter Sentiment Analysis Dashboard
Overview
This Power BI dashboard analyzes Twitter data related to **Call of Duty** to understand how people feel about the game.  
It shows key insights like positive, negative, and neutral tweet counts, average engagement, and sentiment score distribution.
